# Screenshot Guide

Use this folder for README preview images so the project page stays clean and consistent.

## Recommended Images

- `home.png` - home screen with film selection
- `live-capture.png` - countdown/live capture moment
- `final-strip.png` - final strip + action buttons

## Capture Tips

- Use desktop browser at full window size
- Keep the booth centered in frame
- Avoid browser UI overlays in the screenshot
- Capture in PNG for sharper text and edges

## Suggested Dimensions

- Preferred width: **1400 px**
- Aspect ratio: **16:10** or **16:9**
- Keep all screenshots the same size for a neat README layout

## Optional Export Command (ImageMagick)

If you want to normalize all images to the same width:

```bash
magick input.png -resize 1400x output.png
```

## Naming Convention

Use lowercase and hyphens:

- `home.png`
- `live-capture.png`
- `final-strip.png`
